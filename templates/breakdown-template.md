# Breakdown Template

Use this template for structural and competitive analysis of videos and articles.

---

## Output Format

```markdown
# Breakdown: [Title]

- **Source:** [url or type]
- **Duration/Length:** [estimate]

## Structure

### Hook

- **Technique:** [type — question, bold claim, story, statistic, etc.]
- **Effectiveness:** [high/medium/low + why]

### Content Format

- **Type:** [listicle / tutorial / story / problem-solution / interview / case study]
- **Sections:** [list with timestamps/positions]

### Engagement Techniques

- [Technique]: [example from content]

## Communication Style

- **Tone:** [description]
- **Teaching approach:** [description]

## Strengths

- [What works well]

## Opportunities

- [What could be improved]
```

## Notes

- For articles: analyze headline, intro hook, section structure, CTAs, SEO patterns.
- For videos: analyze pacing, visual cues, retention hooks, CTA placement.
- Be specific in Strengths/Opportunities — cite examples from the content.
- Engagement Techniques should list concrete tactics (pattern interrupts, open loops, social proof, etc.).
